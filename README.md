# uuid64
Generate a 64-bit encoded uuid in Ruby with length of 22.

## Status
[![Gem Version](https://badge.fury.io/rb/uuid64.svg)](https://badge.fury.io/rb/uuid64)

## Get Started

### Install
```bash
gem install uuid64
```

### Using Bundler
```ruby
gem 'uuid64'
```

### Generate
```ruby
require 'uuid64'
puts SecureRandom.uuid64
```
